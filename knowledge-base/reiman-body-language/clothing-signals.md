# Одежда и аксессуары как невербальные сигналы

**Источник:** Тоня Рейман «Законы привлекательности. Язык тела для женщин», Питер, 2012, Глава 17

---

## Базовые принципы

- Одежда — неотъемлемая часть бессловесного языка тела; посылает сигналы о образовании, вероисповедании, личности и статусе.
- Люди бессильны против впечатления, которое производит одежда; суждения происходят автоматически.
- Важно одеваться в соответствии с внутренними намерениями: «рабочий» стиль ≠ «свидание» ≠ «вечеринка».

---

## Цвет как сигнал намерений

| Цвет | Сообщение / Эффект | Рекомендация |
|------|-------------------|--------------|
| **Красный** | Страсть, сексуальные намерения, возбуждение; учащает сердцебиение наблюдателя | Использовать для усиления внимания; мужчины особенно реагируют на красный |
| **Жёлтый** | Привлечение внимания, настойчивость, бунтарство (в сочетании с чёрным) | Для выделения из толпы; первый цвет, который замечают |
| **Оранжевый** | Страсть + внимание; сигнал экспериментов, необузданности | Для демонстрации смелости и открытости к новому |
| **Белый** | Доступность + неопытность в сексе; невинность | Для создания образа «не тронутой»; усиливается румянцем |
| **Розовый** | Страсть красного + невинность белого; замедляет выброс адреналина | Самый выгодный цвет для романтических встреч |
| **Синий** | Уверенность, холодность, решительность | Не для демонстрации доступности; для первого шага |
| **Зелёный** | Спокойствие, гармония, естественность | Для обращения к размеренным, не суетливым людям |
| **Фиолетовый** | Меланхолия; страсть подавлена холодом | Насыщенный = скрытые желания; светлый = как розовый |
| **Коричневый** | «Скучный», «загрязнённый» сигнал | Только если есть другие уникальные черты; иначе — незаметность |
| **Чёрный** | Скрывает недостатки, но и лицо; незаметность в толпе | Оставлять для похорон или «основы» под яркий верх |

*Примечание: Яркие цвета привлекают внимание по эволюционной причине — глаза учились находить еду; нейтральные цвета «не напоминают еду» и игнорируются.*

### Правило «сливочного мороженого»
- Нижняя половина (основа) — нейтральный цвет («мороженое»)
- Верх — яркий акцент, передающий настроение и намерения («вишня»)
- Такая комбинация цепляет взгляды, как оперение павлина

## Определение своего «сезонного» цвета

| Тип | Характеристики | Подходящие цвета | Избегать |
|-----|---------------|-----------------|----------|
| **Зима** | Голубой/розовый оттенок кожи; тёмные волосы/глаза или платиновые волосы + светлые глаза | Чёрный, белый, тёмно-синий, красный, голубой, розовый, жёлтый | Золотой, оранжевый, бежевый |
| **Лето** | Как зима, но чаще светлые волосы; слабый контраст кожа/волосы/глаза | Коричневатый, розовый, розовато-лиловый, бледно-лиловый, зеленовато-голубой, сливовый | Чёрный, оранжевый, яркие краски |
| **Осень** | Золотистый/персиковый оттенок кожи; рыжие/каштановые волосы; тёмные глаза с золотистым отливом | Оливковый, бежевый, золотой, оранжевый, коричневый, серый | Чёрный, белый, синий, пастель |
| **Весна** | Золотистые оттенки, светлая кожа; волосы от пшеничного до рыжего; светлые глаза, веснушки | Персиковый, оранжево-розовый, золотисто-жёлтый, коричневатый, сине-зелёный, красный, голубой | Чёрный, белый, очень тёмные цвета |

## Косметика и макияж
| Элемент | Цель / Эффект |
|---------|--------------|
| **Красная помада** | Отражает возбуждённые половые губы; усиливает сексуальное внимание; не для тех, кто избегает такого внимания |
| **Подводка/тушь** | Выделяет глаза — женственную черту; отвлекает от мужских черт лица |
| **Румяна** | Моделируют розовые щёки — признак женственности и здоровья |
| **Тональный крем** | Выравнивает тон, визуально уменьшает нос и подбородок (мужские черты) |
| **Утончение бровей** | Уменьшает типично мужскую черту; делает лицо более женственным |

*Примечание: Косметика усиливает эффекты эстрогена (большие глаза, губы, щёки) и минимизирует эффекты тестостерона (крупные черты).*

## Драгоценности и аксессуары

| Тип | Сообщение |
|-----|-----------|
| **Сердцевидные украшения** | Романтизм; ностальгия по прошлой любви; избегать, если хотите казаться открытой |
| **Крупные, кричащие украшения** | Сильная личность, власть; могут подавлять скромных женщин |
| **Изящные, миниатюрные украшения** | Простодушие, ранимость, доступность |
| **Асимметрия в украшениях** | Желание распространять гены, игривость; возбуждает интерес наблюдателя |

## Коррекция фигуры через одежду

| Особенность | Решение |
|-------------|---------|
| **Широкие плечи** | Глубокий неширокий вырез (вертикальный); маленькие/отсутствующие воротники; ниспадающие рукава; тёмные цвета выше пояса, светлые — ниже; избегать наплечников, горизонтальных полос |
| **Узкие бёдра / толстая талия** | Юбки/платья, расширяющиеся от талии; складки, горизонтальная полоска ниже пояса; утяжка на поясе; контрастные цвета блузки и юбки; брюки с боковыми карманами |
| **Низкий рост** | Шарфики, шляпки, броские украшения на шее; одноцветная одежда; топики до бёдер; заправлять рубашку; прямые юбки; мини или брюки в пол; вертикальные полосы; туфли в тон брюкам/юбке; каблуки; ниспадающие волосы от макушки; V-образные ожерелья |
| **Маленькая грудь** | Глубокий/средний вырез; кружева, сборка на груди; горизонтальные линии над грудью; затянутая талия; светлые топы + тёмный низ; одежда своего размера |
| **Плоские ягодицы** | Объёмное нижнее бельё; плотно прилегающие джинсы; карманы с клапанами; украшения на задних карманах; широко расставленные карманы |

## Ткани, покрой, детали

| Элемент | Эффект / Рекомендация |
|---------|----------------------|
| **Текстурированные ткани** (сатин, шёлк, кашемир, замша) | Вызывают желание прикоснуться; усиливают тактильный интерес |
| **Колышущиеся платья / широкие штаны** | Усиливают женственность; покачивание при походке увеличивает привлекательность до 50% |
| **Открытая обувь** (сандалии, плетёнки) | Сигнал желания обнажиться; привлекают внимание к лодыжке, пальцам, ложбинке между пальцами |
| **Узкие туфли / с завязками** | Подсознательный сигнал готовности быть «связанной»; надежда на секс |
| **Высокие каблуки** | Удлиняют лодыжки и ноги; увеличивают ягодицы визуально на ~25%; привлекают внимание; жертва комфорта ради сексуальности — сигнал для мужчины |
| **Ямочка на шее (надгрудинная)** | Уязвимое место; открытый вырез или округлое ожерелье с кулоном на ямке — сигнал открытости |
| **Открытые плечи** | Эмоциональная реакция очевиднее; женский контур (имитация груди) обозначает половое различие; тонкие лямочки привлекают внимание к плечам |
| **Татуировки / пирсинг** | Сигнал «не как все», таинственность; но могут восприниматься как бунт или агрессия; начинать с малого, скрываемого; не демонстрировать публично без цели |
| **Асимметрия в одежде** (вырез, брошь, одно открытое плечо) | Возбуждает интерес; симметрия лица = хорошие гены, асимметрия в стиле = желание «распространять гены» или повеселиться |
| **Графические изображения** (линии, фигуры, брошки) | Радуют визуальные центры мозга; привлекают взгляд |

## Джинсы: универсальный сигнал

- Синий цвет — один из трёх основных, к которым особенно восприимчив человеческий глаз
- Тёмные джинсы = сдержанность; светлые = спокойствие
- Карманы с украшениями/швами привлекают внимание к ягодицам
- Контрастные нитки по швам визуально удлиняют ноги
- Застёжка-молния спереди привлекает внимание к интимной зоне
- Женский крой подчёркивает изгибы, бёдра, ширину таза
- Хлопок — дышащая ткань, не мешает распространению феромонов

## Практические правила

1. **Одежда своего размера** — стройнит; слишком большая = «сиамские близнецы на грудной клетке»; слишком маленькая = нет простора для фантазии
2. **Соответствие месту** — мини-юбка неуместна на проповеди; водолазка — на красной дорожке
3. **Соответствие намерениям** — если идёте «просто выпить кофе», не одевайтесь как для «охоты»; иначе результат будет одинаковым
4. **Принцип «тёмный низ — яркий верх»** — как у павлина; цепляет взгляды, первый шаг к привлечению

> *Данные извлечены строго из источника. Одежда — первый «язык», который «читают» другие; осознанный выбор усиливает эффективность флирта.*
