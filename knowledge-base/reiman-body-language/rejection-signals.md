# Сигналы отказа и безразличия

**Источник:** Тоня Рейман «Законы привлекательности. Язык тела для женщин», Питер, 2012, Глава 11

---

## Семь универсальных выражений лица (П. Экман)

*Независимы от культуры, пола и воспитания. Отражают синхронную работу 43 лицевых мышц.*

| Выражение | Признаки (лицо) | Интерпретация в контексте знакомства |
|-----------|----------------|--------------------------------------|
| **Счастье** | Расслабленный лоб, опущенные брови, суженные глаза, морщинки смеха, приподнятые щеки, видны зубы | Искренний интерес, можно продолжать общение |
| **Удивление** | Приподнятые брови, расширенные глаза, сморщенный лоб, приоткрытый рот | Нейтрально: реакция на вас или на окружение; требует доп. анализа |
| **Страх** | Приподнятые брови, приподнятые верхние веки, неподвижные нижние, напряженные губы, уголки рта опущены | Возможно, вы его смущаете или он встревожен ситуацией; смягчите стиль |
| **Гнев** | Нахмуренные брови, напряженные нижние веки, опущенные верхние, пронизывающий взгляд, расширенные ноздри, поджатые губы | **Стоп**: уходите, не провоцируйте |
| **Раздражение** | Гладкий лоб, приспущенная бровь, сморщенный нос, напряженные нижние веки, сжатые губы | Вы вызываете дискомфорт; отступите |
| **Презрение** | Сморщенный нос, скривленные губы, одна бровь приподнята, ехидная улыбка, верхняя губа приподнята с одной стороны | Вы обидели человека или он считает себя выше; опасно |
| **Печаль** | Опущенные веки, нахмуренный лоб, уголки рта опущены вниз | Не обязательно из-за вас; милая улыбка может улучшить настроение |

*Примечание: Контекст критически важен. Одно выражение может означать разное в зависимости от ситуации.*

## Сигналы отсутствия интереса / отказа

| Категория | Сигнал | Что значит / Как действовать |
|-----------|--------|------------------------------|
| **Взгляд** | Незаинтересованный взгляд (смотрит в сторону или вверх после контакта) | Не хочет играть в вашей команде |
|  | Грубый взгляд / закатывание глаз | Пренебрежение, пассивная агрессия, желание уйти; **бежать** |
|  | Зрачки сужены (при нормальном освещении) | Отсутствие возбуждения/интереса |
| **Поза и движение** | Заскучавший ковбой (осматривает комнату, зевает, подбородок на ладони, смотрит на часы) | Не заинтригован; не делать ставку |
|  | Еле стоит на ногах (вялый, уставший вид) | Сексуально возбужденный мужчина не выглядит уставшим; отступить |
|  | Скрещивание ног + сжатие ног | Вероятно, не заинтересован в дальнейшем |
|  | «Холодное» бедро (разворачивает бедро как щит) | Защита чувствительных зон; если через 2–5 мин не раскрывается — отступать |
|  | Неприкасаемый (отпрыгивает при попытке касания) | Подсознательный или осознанный отказ от близости |
| **Дистанция** | Установление границ (стоит далеко, как будто «другой почтовый индекс») | Избегает личного пространства; не увлечен |
|  | Блокада (скрещенные руки на груди) | Блокирует восприятие информации; протянуть напиток/меню — если не раскрывается, откланяться |
| **Мимика и речь** | Фальшивая улыбка (рот растянут, но губы сомкнуты, глаза не сощурены, нет морщинок) | Вежливость или страх; не романтический настрой |
|  | Неискренняя улыбка (углы рта приподняты, но другие элементы счастья отсутствуют) | Подозрение на неискренность; осторожность |
|  | Бесцеремонное вмешательство (перебивает, резко меняет тему на свою) | Следующий шаг — смена объекта интереса |
|  | Основные вопросы без страсти | Избегание неловкого молчания, а не интерес |
|  | Мужчина из Энсино (холоден, краткие ответы, закрывается плечом) | Не растопить сердце; отступить |
|  | Ищущий чего-то больше (разглядывает комнату, смотрит на дверь) | Ждет кого-то другого или ищет более интересный контакт |
| **Поведение** | Великое исчезновение / Без вести пропавший | Намеренно смешивается с толпой или уходит; искать другого |

## Смешанные выражения и контекст

- Когда смешиваются два из семи выражений, нужно определить **доминирующую эмоцию**.
- Пример: улыбка + нахмуренные брови + дергающиеся ноздри → попытка скрыть гнев или сильное волнение.
- Пример: улыбающийся рот + печальные глаза → притворное сочувствие, скрытая радость.
- **Правило:** Сосредоточьтесь на эмоции, наиболее нежелательной в текущей ситуации. Если тело говорит «нет», а слова «да» — верьте телу.
- **Коррекция лица:** Если выражение печали/страха временное, можно попробовать поднять настроение милой улыбкой, демонстрацией уязвимых мест или многообещающим взглядом. Любое движение в сторону позитива — хороший знак. Если остается мрачным или переходит в гнев/раздражение — миссия провалена.

## Важные примечания

- Мужчины чаще скрещивают руки на груди, даже если женщина им интересна. Если при скрещенных руках он выпячивает бицепсы — это демонстрация силы, а не отказ.
- Закатывание глаз может быть шутливым или методом самоосуждения для очарования. Оценивайте ситуацию в комплексе.
- На похоронах или в церквях сигналы будут более скрытыми из-за социальных норм.
- Если вы пропустили сигналы отказа, не вините себя. Теперь вы знаете, какие маркеры искать.

> *Данные извлечены строго из источника. Сигналы следует интерпретировать в комплексе, с учетом контекста, места и индивидуальных особенностей.*
