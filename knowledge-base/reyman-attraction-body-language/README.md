# Тоня Рейман «Законы привлекательности. Язык тела для женщин»

**Источник:** Тоня Рейман «Законы привлекательности. Язык тела для женщин», Издательство «Питер», Санкт-Петербург, 2012, 320 с., ISBN 978-5-459-01055-8, тираж 4000 экз.

---

## Информация об авторе

Тоня Рейман — автор книг по психологии отношений, специалист в области невербальной коммуникации и эволюционной психологии. В книге опирается на исследования в области нейробиологии, этологии и социальной психологии.

---

1. Введение: Почему мы выбираем тех, кого выбираем?

11. Прикосновения: как физический контакт усиливает связь
