# Выведение пятен

**Источник:** С.В. Баранова «Волшебный быт», Глава 5

---

## Общие правила

1. **Сохраняй спокойствие** — суета усугубляет загрязнение
2. Сначала удали грубые остатки — промокай (не три!) ваткой или тряпочкой
3. **Свежее пятно легче вывести** — действуй как можно скорее
4. Прочти этикетку: проверь, можно ли стирать изделие
5. Определи природу пятна → выбери тактику
6. Первое средство — **тёплый мыльный раствор + зубная щётка**
7. Любое средство **сначала проверь** на скрытом участке ткани
8. Чтобы не было ореолов — смочи ткань вокруг пятна перед обработкой
9. Обрабатывай пятно **с изнаночной стороны**

---

## Ограничения по веществам

| Вещество | Нельзя применять на |
|----------|-------------------|
| Хлорная известь | Хлопчатобумажные ткани (разрушает) |
| Спирты и кислоты | Некоторые красители |
| Ацетон и уксусная кислота | Ацетатный шёлк |
| Щёлочи и отбеливатели | Только для белых тканей |

## Классификация пятен и способы выведения

| Пятно | Как вывести |
|-------|------------|
| **Жир** (масло, маргарин) | Свежее: прогладь тёплым утюгом через 2–3 слоя бумаги → дочисти бензином. Или: засыпь крахмалом/зубным порошком → бензин/нашатырь |
| **Фрукты, соки, ягоды** | Смочи столовым уксусом → прополощи холодной водой. Свежее: засыпь солью → постирай |
| **Жевательная резинка** | В морозильник на 1 час → снять тупым ножом → протереть ватой в бензине/спирте/ацетоне |
| **Йод** | Смочи холодной водой → протри крахмалом до исчезновения → постирай |
| **Какао, шоколад** | Постирай в сильно подсоленной воде или растворе нашатыря |
| **Клей** | Промокни ацетоном или бензином |
| **Косметика, духи** | Бензин |
| **Краска для волос** | Перекись водорода + нашатырный спирт (равные части) |
| **Кровь** | Только холодная вода! Горячая — закрепляет навсегда. Замочить в холодной воде → тёплый мыльный раствор. Шерсть: тёплая вода с растворённым аспирином → высушить феном |
| **Лак для ногтей** | Жидкость для снятия лака с изнанки (подложить бумажную салфетку) → постирать |
| **Масляная краска** | Свежая: скипидар/ацетон/керосин → нашатырный спирт. Застарелая: скипидар → крепкий раствор соды → тёплая вода |
| **Ржавчина** | Свежая: лимонный сок → прогладить через влажную тряпку. Застарелая: раствор уксусной кислоты (2 ст.л. на стакан воды) при 80–90°C → промыть нашатырём (1 ст.л. на 1 л воды) |
| **Свечной воск** | Соскоблить тупым ножом → несколько слоёв бумаги → прогладить тёплым утюгом (менять бумагу) |
| **Трава, листья** | Тёплый мыльный раствор + нашатырь. Свежие: горячая вода с солью. На цветных тканях: крепкий раствор соли или денатурированный спирт |
| **Тушь, гуашь** | Только холодная вода с моющим средством. Или замочи в холодном растворе → после высыхания почисти щёткой |
| **Фломастер** | Под пятно бумажную салфетку → промокни чистым спиртом |
| **Цветочная пыльца** | Промокни спиртом → постирай |
| **Шариковая ручка** | Промокни спиртом → постирай |
| **Залоснённые места** (воротники, рукава) | Почисти щёткой от пыли → протри бензином/ацетоном → оставшееся — нашатырь 1:10 с водой |
| **Пот** | Свежие: крепкий раствор соли → постирать. Шерсть: бензин. Шёлк: нашатырь + денатурат. Белый шёлк: перекись с водой |
| **Вода (следы дождя) на шёлке** | Опрыскать солёной водой → высушить |
| **Шерстяные вещи** | Чистить щёткой с водой или гладить через влажную ткань |

## Для успешной борьбы с пятнами держи дома

- Пятновыводитель универсальный
- Отбеливатель (только для белых тканей!)
- Бензин (очищенный, для чистки)
- Нашатырный спирт
- Уксус столовый
- Перекись водорода
- Ацетон
- Зубной порошок / крахмал
- Жидкость для снятия лака
