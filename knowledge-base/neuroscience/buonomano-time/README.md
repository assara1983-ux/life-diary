# Нейробиология времени — Д. Буономано, «Мозг – повелитель времени»

> ⚠️ **Важно**: Это не астрологический источник. Данные основаны на экспериментальной нейробиологии и не персонализируются по дате рождения.

## Назначение
Предоставить ИИ-советнику научно обоснованные алгоритмы для:
- Оптимизации планирования с учётом циркадных ритмов
- Коррекции субъективной оценки времени пользователем
- Поддержки межвременного выбора (immediate vs delayed reward)
- Использования эпизодической памяти для симуляции будущего

## Принципы интеграции
1. **Алгоритмичность**: каждый раздел содержит псевдокод или чёткие правила `if/then`
2. **Контекстная привязка**: рекомендации мапятся на сущности Life Diary (`Work`, `Goals`, `Journal`, `Today`)
3. **Неперсонализация по дате рождения**: адаптация происходит на основе поведенческих данных пользователя, а не астрологических расчётов
4. **Изоляция от астрологии**: папка `neuroscience/` логически отделена от `astrology/`

## Источники данных для персонализации
- История выполнения задач (фактическое время vs запланированное)
- Журнал настроения и уровня энергии (если ведётся)
- Паттерны сна/активности (если интегрировано с трекерами)
- Явные предпочтения пользователя («я жаворонок», «вечером мне легче концентрироваться»)

## Лицензия
Книга распространяется в рамках проекта «Дигитека» / «Всенаука». Использование — только для персонального некоммерческого применения в рамках проекта Life Diary TMA.
