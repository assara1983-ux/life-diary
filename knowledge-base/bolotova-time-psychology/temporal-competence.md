# Временная компетентность: определение и компоненты

**Источник:** Болотова А.К. «Психология организации времени», Аспект Пресс, Москва, 2006, Глава 4.1

---

## Определение

**Временная компетентность** (компетентность во времени) — способность адекватно оценивать временные ресурсы, умение расширять или сужать временные рамки межличностных взаимодействий, своевременно делегировать полномочия в целях экономичного и рационального использования времени.

> *«Временная компетентность человека предполагает наличие некоторой совокупности знаний и умений, связанных с ощущением временного пространства и временных лимитов общения, "чувства времени"»*.

---

## Компоненты временной компетентности

### 1. Когнитивный компонент
| Умение | Описание |
|--------|----------|
| Адекватное восприятие времени | Точная оценка длительности интервалов, чувство темпа и ритма |
| Прогнозирование временных перспектив | Способность предвидеть сроки, планировать этапы деятельности |
| Знание стратегий планирования | Владение методами тайм-менеджмента, приоритизации |

### 2. Эмоционально-ценностный компонент
| Умение | Описание |
|--------|----------|
| Ценностное отношение ко времени | Осознание необратимости, невозможности накопления времени |
| Эмпатия к времени другого | Уважение к чужому времени, пунктуальность |
| Регуляция тревожности при дефиците времени | Сохранение эффективности в условиях цейтнота |

### 3. Поведенческий компонент
| Умение | Описание |
|--------|----------|
| Рациональное распределение приоритетов | Выделение задач А/Б/В, принцип Эйзенхауэра |
| Делегирование полномочий | Передача задач с сохранением контроля сроков |
| Организация временной структуры деятельности | Последовательность операций, ритм, паузы |

---

## Уровни временной компетентности (по В.И. Ковалеву)

| Тип | Характеристика | Пример поведения |
|-----|---------------|-----------------|
| **Стихийно-обыденный** | Зависимость от обстоятельств, пассивность, отсутствие инициативы | Реагирует на события постфактум, не планирует |
| **Функционально-действенный** | Избирательная активность, охватывает отдельные периоды | Планирует важные задачи, но не прогнозирует долгосрочно |
| **Созерцательно-пролонгированный** | Пассивное отношение, хаотичность, отсутствие чёткости | Имеет общие представления о будущем, но без конкретики |
| **Созидательно-преобразующий** | Организация времени во взаимосвязи со смыслом жизни, творческое наполнение | Активно формирует жизненные стратегии, резервирует время для развития |

---

## Типология личностной организации времени (по Л.Ю. Кублицкене)

| Тип | Характеристика |
|-----|---------------|
| **Оптимальный** | Успешно осваивает любые временные режимы, решает любые временные задачи |
| **Дефицитный** | Субъективно сводит все режимы к дефициту времени, но выполняет задачи успешно |
| **Спокойный** | Испытывает трудности только в режиме дефицита времени, в остальных — успешен |
| **Исполнительный** | Успешен во всех режимах, кроме временной неопределённости; преобладает чувство порядка |
| **Тревожный** | Успешен при избытке времени, крайне непродуктивен в условиях дефицита |
| **Неоптимальный** | Не успешен ни в одном режиме; конфликтный характер переживания времени |

## Индивидуальный t-тип восприятия времени

Формула: **t = t<sub>с</sub> / t<sub>о</sub>**, где  
- t<sub>с</sub> — время, воспроизведённое испытуемым  
- t<sub>о</sub> — задаваемая длительность

| Тип | Диапазон t | Характеристика |
|-----|-----------|---------------|
| **Спешащий** | 0,7 ≤ t < 1,0 с | Субъективное время опережает объективное; постоянный дефицит времени, устремлённость в будущее |
| **Точный** | t = 1,0 с | Субъективная единица синхронизирована с объективной; привязан к реальному времени |
| **Медлительный** | t > 1,0 с | Субъективное время отстаёт от объективного; время «движется медленно», ориентация на прошлое |

> *Примечание: t-тип — жёсткая психологическая константа, не меняется в течение жизни (Б.И. Цуканов).*
