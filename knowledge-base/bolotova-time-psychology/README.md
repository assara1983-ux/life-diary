# Психология организации времени — База знаний

**Источник:** Болотова А.К. «Психология организации времени», Аспект Пресс, Москва, 2006, 254 с., ISBN 5-7567-0420-5

**Об авторе (из книги):**  
Алла Константиновна Болотова — доктор психологических наук, профессор, специалист в области психологии времени, организации деятельности и коммуникативной компетентности. Книга рекомендована Советом по психологии УМО по классическому университетскому образованию как учебное пособие для студентов направления «Психология».

**Структура книги:**
| Глава | Название |
|-------|----------|
| Введение | Актуальность проблемы времени в психологии |
| Глава 1 | Человек и время |
| Глава 2 | Психология организации времени в деятельности человека |
| Глава 3 | Общение и его организация во времени |
| Глава 4 | Тайм-менеджмент и развитие временной компетентности |
| Глава 5 | Методы изучения временных восприятий и компетентности во времени |

**Что не извлечено:**
- Некоторые таблицы в Главе 5 (методики) частично нечитаемы из-за качества скана — помечено в соответствующих файлах
- Экспериментальные данные (числовые значения, графики) приведены в описательной форме, так как визуальные элементы не распознаны

**Разделы приложения, использующие эту книгу:**
| Раздел | Как используется |
|--------|-----------------|
| **Сегодня** | Рекомендации по планированию дня с учётом типа восприятия времени, чувство «здесь и теперь» |
| **Расписание** | Приоритизация задач (АБВ-анализ), принцип Эйзенхауэра, резервирование времени |
| **Работа** | Профессиональная адаптация, делегирование, анализ временных потерь |
| **Здоровье** | Временные циклы работоспособности, фазы утомления, профилактика стресса |
| **Ментальное** | Самооценка, временные перспективы, техники совладания с кризисами |
| **Цели** | Разведение реальных и идеальных целей, целеполагание во времени |
| **Профиль** | Диагностика временной компетентности, t-тип, тип организации времени |
| **Журнал** | Рефлексия событий во времени, анализ прошлого опыта |
