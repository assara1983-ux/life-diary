## Анкета самооценки использования времени

**Инструкция:** Оцените по шкале:  
0 = почти никогда, 1 = иногда, 2 = часто, 3 = почти всегда

| № | Утверждение | Балл |
|---|-------------|------|
| 1 | Резервирую в начале рабочего дня время для подготовительной работы, планирования | |
| 2 | Перепоручаю все, что может быть перепоручено | |
| 3 | Письменно фиксирую задачи и цели с указанием сроков их реализации | |
| 4 | Каждый официальный документ стараюсь обработать за один раз и окончательно | |
| 5 | Каждый день составляю список предстоящих дел, упорядоченный по приоритетам | |
| 6 | Свой рабочий день по возможности освобождаю от посторонних телефонных разговоров, незапланированных посетителей | |
| 7 | Свою дневную нагрузку стараюсь распределить в соответствии с графиком моей работоспособности | |
| 8 | В плане моего времени всегда есть «окна», позволяющие реагировать на актуальные проблемы | |
| 9 | Я пытаюсь направить свою активность таким образом, чтобы в первую очередь концентрироваться на немногих, жизненно важных проблемах | |
| 10 | Я умею говорить «нет», когда на мое время претендуют другие, а мне необходимо выполнить более важные дела | |

**Интерпретация суммы баллов:**
| Диапазон | Характеристика |
|----------|---------------|
| 0–15 | Не планируете время, находитесь во власти внешних обстоятельств |
| 16–20 | Пытаетесь овладеть временем, но не всегда последовательны |
| 21–25 | Довольно рационально используете время; важно правильно выбирать приоритеты |
| 26–30 | Образец рационального расходования времени |

## Метод «Альпы»: 5 стадий рационализации времени

> *«Метод относительно прост, и после некоторых упражнений потребуется в среднем не более 10 минут для составления ежедневного плана»*.

| Стадия | Действие | Инструмент |
|--------|---------|-----------|
| **А** — Составление заданий | Выписать все дела на день | Список, стикеры, приложение |
| **Л** — Оценка длительности | Прикинуть время на каждое дело | Опыт, аналогии, буфер +20% |
| **П** — Резервирование (60:40) | Запланировать только 60% времени | Календарь с «окнами» |
| **Ы** — Принятие решений по приоритетам | Категории А/Б/В, делегирование | Матрица Эйзенхауэра |
| **С** — Контроль (учёт несделанного) | Вечерний анализ, перенос | Дневник, рефлексия |

## Рациональное использование рабочего дня: чек-лист

### Утро
- [ ] Начинать день с позитивным настроением, без спешки
- [ ] Приступать к работе в одно и то же время
- [ ] Пересмотреть план дня, внести коррективы
- [ ] Самое продуктивное время — ключевым задачам (категория А)

### В течение дня
- [ ] Отклонять дополнительные неотложные проблемы (делегировать)
- [ ] Избегать незапланированных импульсивных действий
- [ ] Делать паузы: 1 час работы → 10 минут отдыха
- [ ] Однородные задачи выполнять сериями
- [ ] Использовать незапланированные паузы для мелких дел
- [ ] Выкраивать «закрытый час» для важных дел / творчества / самообразования

### Вечер
- [ ] Завершить несделанное или перенести с пометкой
- [ ] Просмотреть план, подвести итоги
- [ ] Составить план на следующий день

## Принципы планирования: резюме
Планируйте лишь 60% времени (основное соотношение 60:40)
✓ Фиксируйте в планах конечные результаты, а не просто действия
✓ Задавайте точные временные нормы (опыт: работа занимает всё отведённое время)
✓ Работайте над планами регулярно, системно, последовательно
✓ Стремитесь сразу восполнять потери времени
✓ Используйте письменную форму — это повышает обязательность
✓ Переносите несделанное осознанно, а не «забывайте»
