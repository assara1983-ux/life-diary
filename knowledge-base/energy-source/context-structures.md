# Контекст, видение, системы и структуры

**Источник:** «Источник Энергии», МИФ, 2012, Главы 7–9

---

## Три слагаемых успеха

```
Контекст (карта и маршрут)
    +
Энергия (топливо)
    +
Системы и структуры (транспорт)
    =
Результат
```

**Контекст** — твоё «ЧТО» и «ПОЧЕМУ». Видение, мотивы, убеждения, мировоззрение. Очерчивает сферу возможностей.

**Энергия** — топливо для движения. Определяет, с какой скоростью продвигаешься к цели.

**Системы и структуры** — организационные элементы: расписание, среда, обязательства, люди.

---

## Контекст: видение и цель

**Видение** — чёткая картина жизни или проекта в конкретный момент будущего.
*Пример:* «Через пять лет я стану главой подразделения и буду счастливо жить с семьёй в замечательном пентхаусе в центре Лондона».

**Цель** — конкретный результат, который хочешь получить.
*Пример:* «До конца года получить новую должность с повышением зарплаты не менее чем на 5%».

**Сила видения (история о трёх каменщиках):**
- Первый: «Я кладу кирпичи»
- Второй: «Я зарабатываю на жизнь»
- Третий: «Я строю собор»

Третий каменщик, скорее всего, обладает большим запасом жизненной энергии и добьётся большего — потому что его вдохновляет контекст.

## Структуры

**Структура** — элементы организации, которые помогают выполнять обязательства и поддерживают на пути к целям.

### 4 основных элемента структуры

| Элемент | Описание |
|---------|---------|
| **Расписание** | Способ организации времени: внешние факторы (работа, школа) + внутренние (вставать в 7 для спортзала) |
| **Окружающая среда** | Способ организации пространства: офис, кабинет, комната |
| **Обязательства и договорённости** | Содержание работы, встречи, намерения, забота о близких |
| **Люди** | Кто работает рядом, кто поддерживает |

### Примеры структур для конкретных целей

**Похудеть на 3 кг за 2 месяца:**
- Расписание: когда и в какое время заниматься?
- Среда: где (спортзал, дома, на улице)?
- Обязательства: нанять тренера, вести таблицу прогресса
- Люди: партнёр для тренировок

**Структуры для здоровья:**
- Выделить время на спортзал
- Создать таблицу достижений
- Нанять тренера с контролем
- Бегать по утрам
- Убрать из дома нездоровую пищу

**Структуры для отношений:**
- Еженедельное «свидание» с партнёром
- В выходные — время с друзьями
- Ежедневное общение с близким человеком
- Раз в месяц — званый обед для друзей

**Структуры для финансов:**
- Еженедельный/ежемесячный финансовый анализ
- Автоматическое пополнение сберегательного счёта

**Структуры для личного развития:**
- Еженедельный анализ личных и профессиональных целей
- Цель или задача на ближайший месяц

## Структуры, которые НЕ работают

| Структура | Почему ограничивает |
|-----------|-------------------|
| Политика открытых дверей | Много отвлекающих факторов |
| Уведомления о каждом письме | Постоянные отвлечения от задач |
| Доступ в интернет без ограничений | Свойственно отвлекаться |
| Слишком жёсткая структура | Когда нужен гибкий подход |
| Отсутствие структуры | Когда нужны упорядоченность и концентрация |
| Самостоятельное решение всех задач | Когда команда даёт лучший результат |
| Нет ограничения рабочего времени | Работа заполняет всю жизнь |

## Упражнение: внедрение структуры

1. Определи одну область жизни, где хочешь изменений
2. Ответь на 4 вопроса:
   - **Расписание:** Когда и в какое время это делать?
   - **Среда:** Где это делать? Нужно ли изменить среду?
   - **Обязательства:** Какие могу взять дополнительно?
   - **Люди:** Кто будет вовлечён?
3. Запиши конкретный план
4. Проанализируй через неделю — работает ли структура
